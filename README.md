# trashure

PASTIKAN MENDOWNLOAD NODE.JS DAN MYSQL(XAMPP) 

Frontendnya = Next.js and Material Ui 
Backendya = Node.js dan express
Database = Mysql 

Untuk aktifkan database ke backend/server.js masukan di terminal
node server.js

untuk aktifkan frontend ke frontend/app/pages.tsx masukan ke terminal 
npm run dev
